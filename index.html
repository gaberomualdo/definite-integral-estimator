The True G's Integral Estimator estimate definite integrals like a absolute G.

How It Works

  1. Enter your integral and generate the graph. Then download the graph onto your computer. Take a look at the photo if you want. It should be a photo of a graph of the integral.
  2. Upload the downloaded graph into the estimator section and get your result.
  3. Use any variable name you want but make sure to only have one variable used; this is case sensitive. Use an astericks in place of general multiplication in some cases to be safe. Constants such as PI and E are not supported at this time. Do not change the given name of the file before you upload. Things can get screwed up when a and b are not integers. Make sure a is less than b. Some equations work better than others (in some cases certain equations don't work at all).
  4. If you are legitimately using this calculator to solve definite integrals: why? Are you okay. I am concerned.
